# 📦 Sistema de Control de Inventario - UAPA

## 📝 Descripción del Proyecto
Este sistema surge como solución tecnológica ante la problemática común que enfrentan los negocios locales en Santiago, República Dominicana: la falta de automatización y el registro manual de sus mercancías. La ausencia de un control digital genera pérdidas económicas, descuadres en almacén y retrasos en la toma de decisiones. 

Este software permite digitalizar el flujo operativo, ofreciendo un control preciso sobre las entradas, salidas y existencias reales de productos en tiempo real.

---

## 🎯 Objetivos del Proyecto

### Objetivo General
Desarrollar una aplicación de escritorio para la gestión y control de inventarios, optimizando el registro de mercancías y garantizando la integridad de los datos de stock mediante un entorno seguro y eficiente.

### Objetivos Específicos
* Implementar un módulo CRUD (Crear, Leer, Actualizar, Eliminar) para la administración de productos.
* Desarrollar un sistema de alertas visuales para notificar cuando un producto esté próximo a agotarse (stock mínimo).
* Integrar una base de datos local y ligera que garantice la persistencia de la información sin requerir configuraciones de red complejas.
* Aplicar buenas prácticas de arquitectura de software, organizando el código fuente de manera limpia y modular.
